|                    | innodb                    | myisam                           |
| :----------------: | :------------------------ | :------------------------------- |
|        事务        | 支持                      | 不支持                           |
|        外键        | 支持                      | 不支持                           |
|        索引        | 聚簇索引                  | 非聚簇索引                       |
|         锁         | 行锁、间隙锁等            | 只支持表锁                       |
|       并发性       | 高                        | 差                               |
|      崩溃恢复      | 支持事务提交、回滚        | 数据可能出现不完整、不一致。     |
| 表行数（count(*)） | 不记录                    | 记录                             |
|        文件        | idb文件：索引和数据放一起 | myi和myd文件：索引和数据分开存放 |
