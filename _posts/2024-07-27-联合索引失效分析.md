key: mysql-07
---

```mysql
联合索引(a, b, c)，select a, b, c：
	where a = xx and b = xx and c = xx; // 符合最左原则走索引
	where a = xx and b = xx; // 符合最左原则走索引
	where b = xx and a = xx; // 不符合最左原则，但是有SQL优化器，优化后等同于where a = xx and b = xx;所以走索引。
	where a = xx and c = xx; // 不符合最左原则，但是5.7推出索引下推，所以走索引
	where b = xx; // 不符合最左原则，但是也索引下推，所以走索引
```
